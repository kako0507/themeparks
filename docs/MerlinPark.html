Class: MerlinPark

MerlinPark

Implements the Merlin Park API framework. Thorpe Park, Alton Towers, Chessington etc. use this API framework

new MerlinPark(options)

Create new Merlin Object. This object should not be called directly, but rather extended for each of the individual Merlin parks
Parameters:
Name Type Description
options Object
Properties
Name Type Argument Default Description
deviceTokenCachetime Number <optional>
 86400000 Time to cache device token, in milliseconds (optional)
userIdCachetime Number <optional>
 43200000 Time to cache user ID, in milliseconds (optional)
dataCachetime Number <optional>
 43200000 Time to ride data, in milliseconds (optional)
